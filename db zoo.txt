//sintaxis de alias

SELECT column_name(s)

FROM table_name AS alias_name;


The syntax for column aliases is:


SELECT column_name AS alias_name

FROM table_name;


In our earlier example with the phone number alias, we would write that out like this:


SELECT customer_phone_number_with_area_code AS Phone

FROM Customers;

//termina sintaxis


//sintaxis para delete

DELETE FROM table_name WHERE condition;


For example: to delete a record (row) for the character named Roger Rabbit, you would write the following:


DELETE FROM characters_table WHERE character_name=’Roger Rabbit’;
